Roblox Internship

Summer 2022

During my software engineering internship at Roblox, my project was to build an external login flow for the Roblox Studio application in C++. Essentially, when the embedded login page fails, my project would redirect users to a browser, allow them to login there, and then return them back to the application, which would be ready for their use. This was challenging because client application development was different from the normal full-stack development I was used to. I had to think about protocol schemes (how to launch an application from a browser), OAuth 2.0, sending data between different processes on an OS, and other low level ideas. I really enjoyed this as I learned a lot by diving into a new corner of development. The impact of this is that Roblox Studio, an application with over 10 million users, now supports an external browser-based login, helping the 5% of users whose embedded login was failing to log in. Not only did I learn the cornerstones of application development, but also how a good engineering organization operates, from design and technical documentation, to communication and presentation, to code reviews, and more.
